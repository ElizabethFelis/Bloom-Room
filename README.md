# Bloom Room

## project description

Bloom Room is a simple website-based psychological horror game. You enter a quiet reflection room guided by Elizabeth Felis, also known as Eliza. At first, Eliza seems friendly and helpful, but as you interact with the website, she begins to act like she knows more than she should.

## how to play
Unzip the document, you should get a folder.

Open `index.html` in a browser. This is the English version of the game, Chinese translation available. it's the file named `index-cn.html`


The player can:
- choose reflection cards
- type one word
- hover over or click hidden room objects
- either follow Eliza's instructions or disobey her

There are two main endings:
- obedient ending: the user stays on the normal path.
- disobedient ending: the user keeps opening forbidden areas and reveals what is beneath the surface

## files needed

# important!!!!!!

Make sure these files are in the same folder:

- `index.html`
- `friendlyFelis.jpeg`
- `friendlyFelisNoHand.jpeg`
- `noHandFriendlySilent.jpeg`
- `noHandFriendlySpeak.jpeg`
- `notFriendly.jpeg`
- `evenMoreNotFriendly.jpeg`
- `obedientCreepy.jpeg`

If the image files are missing or renamed, Eliza will not appear correctly.

## controls

The game uses only mouse and keyboard:
- click buttons to choose options
- type one word in the input box
- hover or click the corners of the room
- click "Start over" to replay after an ending

## content warning

This game may contain flashing lights and may track your movements on screen. But don't worry, these are for gaming purposes only. Please do not proceed if you are sensitive to flashing lights.
